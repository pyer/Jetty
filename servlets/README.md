Embedded Jetty servlets example
===============================

This is a maven project, to build it:

    $ mvn clean package

To run the example:

    $ mvn exec:exec
or

    $ java -jar target/servlets-1.0-jar-with-dependencies.jar


(Use <kbd>CTRL</kbd>+<kbd>C</kbd> to stop)

Open your web browser to:

    http://localhost:8080/
